# คู่มือการติดตั้งและใช้งาน Military Law Roleplay

ระบบจำลองสถานการณ์และคลังกฎหมายทหาร สร้างขึ้นด้วย Next.js และ Tailwind CSS

## 1. การติดตั้ง (Installation)
หากยังไม่ได้ติดตั้ง dependencies ให้เปิด Terminal แล้วรันคำสั่ง:
```bash
npm install --force
```
*(ต้องใช้ --force เนื่องจากบางไลบรารียังไม่อัปเดตให้รองรับ React 19 แต่ใช้งานได้ปกติ)*

## 2. การนำเข้าข้อมูลกฎหมาย (Importing Data)
เนื่องจากไฟล์ต้นฉบับเป็น PDF ระบบไม่สามารถดึงข้อมูลออกมาโดยอัตโนมัติได้สมบูรณ์
กรุณาทำตามขั้นตอนดังนี้เพื่อนำข้อความเข้าสู่ระบบ:

1. เปิดไฟล์ `data/laws.ts`
2. คัดลอกข้อความจากไฟล์ PDF ของท่าน
3. นำมาวางในส่วน `content` ของแต่ละมาตรา
   ตัวอย่าง:
   ```typescript
   {
     id: "section-1",
     title: "มาตรา ๑",
     content: "วางข้อความที่คัดลอกมาที่นี่..."
   }
   ```
4. ทำซ้ำสำหรับไฟล์:
   - พระราชบัญญัติว่าด้วยวินัยทหาร พุทธศักราช ๒๕๖๘.pdf.pdf
   - กฏหมายเกิดใหม่.pdf
   - กฎหมายรัฐธรรมนูญ (3).pdf

## 3. การเพิ่มสถานการณ์จำลอง (Adding Scenarios)
ท่านสามารถเพิ่มโจทย์สถานการณ์ใหม่ๆ ได้ที่ไฟล์ `data/scenarios.ts` โดยทำตามรูปแบบที่มีอยู่เดิม

## 4. การรันโปรแกรม (Running)
พิมพ์คำสั่ง:
```bash
npm run dev
```
แล้วเปิด Browser ไปที่ `http://localhost:3000`
